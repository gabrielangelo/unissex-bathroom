# unissex-bathroom

Implementation of the 3-gender unisex bathroom problem.

### Main tools used
* python 3.5.2

### Running
1. make the project git clone:
```
$ git clone https://github.com/gabrielangelo/unissex-bathroom.git
```

2. running the project: 
```
$ make run
```
